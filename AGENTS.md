# Repository instructions

- Public and reusable content only. Never add company names, personal email addresses, credentials, internal repository names, or absolute home-directory paths.
- Skill directory names use kebab-case and equal frontmatter `name`.
- Skill frontmatter contains exactly `name` and `description`.
- Descriptions state purpose, positive triggers, and a negative boundary against the nearest skill.
- Keep `SKILL.md` under 500 lines; move optional detail to `references/`.
- Run `ruby scripts/validate.rb` before reporting completion.
